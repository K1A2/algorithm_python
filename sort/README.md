# 정렬(Sort) 알고리즘
데이터를 순서대로 나열하는 알고리즘.

## 정렬 알고리즘 종류

### 선택 정렬
* 현재 위치에 들어갈 값을 찾아 정렬하는 방법.
* 가장 작은 값을 앞으로 보내는 과정을 반복하여 정렬한다.
* 앞에서부터 정렬해 모든 값을 순서대로 비교해 본다.
* 시간복잡도: O(n<sup>2</sup>)

### 삽입 정렬
* 데이터를 하나씩 확인하며 적절한 자리에 삽입하는 방법.
* 정렬이 거의 되어있는 상황에서는 매우 빠른 정렬 방법이다.
* 시간복잡도: O(n<sup>2</sup>)
